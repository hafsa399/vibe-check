# vibe checking (first js APP)

This is a simple semester  web project made using **HTML, CSS, and JavaScript**.
It asks you 5 multiple-choice questions and tells you your "vibe" based on your answers.

## How it works

* You answer **5 MCQs**
* Each question has **4 options (a, b, c, d)**
* Every option gives a certain score
* At the end, your total score is calculated
* Based on your score, your **vibe** is shown (like chill, energetic, creative, etc.)

## Features

* Clean and simple design
* Easy to use interface
* Instant result after completing the quiz
* Fun way to explore your personality

## Technologies used

* **HTML** → structure of the page
* **CSS** → styling and layout
* **JavaScript** → logic, scoring, and result generation

## Purpose

This project is just for fun and practice. It helps in learning basic web development and how user input can be used to create interactive experiences.
